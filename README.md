# Messenger Firebase Chat App

Features

- Google Sign In
- Email/Pass Registration / Log In
- Photo Messages
- Video Messages
- Real Time Conversations
- Location Messages
- Search for Users
- Deleting Conversations
- User Profile
- Dark Mode Support
